# Application React Todo List

Cette application React Todo List est conçue pour vous aider à gérer vos tâches de manière simple et efficace. Elle offre les fonctionnalités suivantes :

- **Barre de recherche :** Vous pouvez facilement accéder à une tâche spécifique en utilisant la barre de recherche. Il vous suffit de saisir un mot-clé ou une partie du nom de la tâche, et la liste des tâches sera filtrée en temps réel pour correspondre à votre recherche.

- **Ajouter des tâches :** Vous pouvez rapidement ajouter de nouvelles tâches à votre liste. Il vous suffit de cliquer sur le bouton "Ajouter une tâche", de saisir le nom de la tâche et de la sauvegarder. La nouvelle tâche apparaîtra instantanément dans la liste.

- **Modifier les états des tâches :** Vous avez la flexibilité de définir l'état de chaque tâche. Vous pouvez choisir parmi les états suivants : "En cours", "Fait", "À faire" ou "En retard". Cela vous permet de suivre facilement le statut de chaque tâche.

- **Supprimer des tâches :** Si une tâche n'est plus pertinente ou doit être supprimée, vous pouvez la supprimer en un seul clic. Cette action est réversible, alors ne vous inquiétez pas de la suppression accidentelle.
